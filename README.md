# 심연의 던전 — Abyss Dungeon

Dark and Darker 스타일의 1인칭 던전 탈출 로그라이크. 브라우저에서 바로 실행되는 단일 HTML 파일 게임 (Three.js).

아이템을 파밍해 점점 강해지고, 층마다 보스를 처치하거나 룬 퍼즐을 풀어 봉인된 계단을 열고 더 깊이 내려가라. 죽음은 영원하다 — 최고 도달 깊이만이 기록으로 남는다.

## 플레이

서버 없이 `index.html`을 열면 바로 실행된다. 로컬에서 띄우려면:

```
npx http-server -p 8377
```

## 조작

| 키 | 동작 |
|---|---|
| WASD / Shift / Space | 이동 / 질주 / 점프 |
| 마우스 | 시점 · 좌클릭 공격 (활: 누르고 당겼다 놓기) · 우클릭 막기 |
| E | 상호작용 (상자 · 줍기 · 계단 · 룬 · 상인) |
| Tab | 인벤토리 |
| 1 2 3 | 무기 교체 |
| C | 무기별 특수 공격 (쿨다운 5초) |
| Q / F | 물약 / 붕대 |
| M | 음소거 |

## 시스템

- **절차 생성 던전** — 층마다 새로운 방·복도 배치, 깊이에 따라 테마 변화
- **무기 6종 × 등급 5단계** — 단검(배후 치명타) · 도검 · 장검 · 전투도끼 · 장창 · 장궁, 일반~전설 등급
- **특수 공격** — 연속 찌르기 · 연속 베기 · 대상해 · 회전 베기(360°) · 돌진 찌르기 · 삼중 사격
- **층 진행** — 보스 4종 순환(광역 슬램 · 소환 · 점멸 · 돌진 패턴), 3의 배수 층은 룬 순서 퍼즐
- **파밍 & 경제** — 상자(미믹 주의) · 적 드롭 · 정예 확정 드롭 · 매 층 상인에게서 구매/판매
- **하드코어** — 자연 회복 없음, 회복은 시간이 걸림, 막기는 스태미나 소모, 죽으면 1층부터

## 기술

- Three.js r158 (로컬 번들, 외부 의존성 없음)
- WebAudio 절차 생성 사운드 (앰비언트 드론 · 바람 · 보스전 드럼 · 전 효과음)
- 의존성/빌드 없음 — 정적 파일 그대로 배포
